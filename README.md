## 🚀 Overview | نبذة عن المشروع
هذا المشروع عبارة عن عميل لوجستي ذاتي التشغيل (Autonomous Logistics Agent) مدعوم بقدرات الذكاء الاصطناعي المتقدمة من **Google Gemini 1.5 Pro**. صُمم النظام لتحليل سلاسل الإمداد المعقدة، التنبؤ بالمسارات، وتوفير رؤى استراتيجية فورية بناءً على البيانات اللوجستية المرفوعة.

Enterprise-grade Autonomous Logistics Agent powered by **Google Gemini 1.5 Pro**. It analyzes complex supply chain data and provides real-time intelligent insights for efficient logistics management and optimization.

---

## ✨ Key Features | المميزات الرئيسية
* **Smart AI Agent:** Powered by Gemini 1.5 Pro reasoning for intelligent decision-making.
* **Interactive Dashboard:** Dynamic logistics analytics built with Streamlit for real-time tracking.
* **Data Visualizations:** Interactive charts and graphs powered by Plotly & Pandas.
* **Route Optimization:** Intelligent suggestions for reducing delivery times and costs.
* **Cybersecurity Mindset:** Built with secure coding practices and enterprise-grade protocols.

---

## 🛠️ Tech Stack | التقنيات المستخدمة
* **AI Core:** Google Gemini 1.5 Pro (API)
* **Backend:** Python / FastAPI
* **Frontend & UI:** Streamlit Cloud
* **Data Engine:** Pandas, NumPy & Plotly Express
